# Hex

The sixth generation of the Hex engine.


## Getting Started

### Installation

From PyPI: Coming soon.


### Local Install

```bash
git clone github.com/pagekey/hex
cd hex
uv run hex --help
```


### Run Your First Graph

Run a test DAG with:

```bash
uv run hex run core.main -w tests/integration/ws_simple/
```

Run the test suite with:

```bash
uv run pytest
```
